# Customer Segmentation using K-Means Clustering

## Objective
To segment customers based on annual income and spending score using
K-Means clustering.

## Algorithm
K-Means Clustering (Unsupervised Learning)

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Outcome
Customers are grouped into distinct clusters, helping understand
purchasing behavior.
